Projects

Working Linear Induction Motor – Smith Engineering Hyperloop

What? Worked with a small team of engineers to design, model, and test the first prototype linear induction motor (LIM) for the Smith Engineering Hyperloop design team.

How? Used COMSOL Multiphysics to model the moving magnetic field and electromagnetic behaviour of the LIM, and SolidWorks to develop the complete mechanical and structural design prior to fabrication. Built and tested a scaled prototype on a flywheel test rig to experimentally measure motor efficiency, then used these results to scale the design to a full pod‑sized LIM configuration.

Results

Won first place in the propulsion category at Hyperloop Global 2024.

Autonomous Medical Delivery Robot – Mechatronics and Robotics Design II

What? Collaborated with a team of 4 to design an autonomous delivery robot intended to safely enclose and deliver biohazardous materials within a hospital or lab facility with a regulation biohazard containement box. The system needed to navigate pre-mapped indoor paths, avoid stationary obstacles, and localize accurately despite wheel slippage.

How? Implemented a dual-controller architecture using an Arduino for low-level motor control and encoder feedback, and a Raspberry Pi for high-level navigation and sensor processing. Developed a C++ weighted graph with Dijkstra’s algorithm for shortest-path planning on a pre-mapped indoor network. Integrated encoder-based odometry and LiDAR sensing to enable accurate localization and real-time avoidance of dynamic obstacles, including pedestrians. Designed a model of air-tight biohazard containement box on Onshape.

Results

Achieved functional navigation, localization subsystems and wireless prototype interface on Arduino. (See attachments for file contents.)

Functional Model Elevator – Data Structures and Algorithms

What? Designed, built, and programmed a functional model elevator that leverages data structures to efficiently manage user requests for pick-ups and drop-offs.

How? Implemented two doubly linked lists to track active elevator requests and a separate node for priority users, optimizing efficiency in request handling. Designed the model elevator using Onshape. Developed the hardware system including a 1-inch LCD display, a motor driver to control elevator movement, and a magnetic encoder to accurately track the elevator’s position.

Results

  • All implemented data structures and algorithms functioned as expected for Call, Delete Call, and Priority Requests.
  • Mechanical components performed reliably: functional floor buttons, priority button, and smooth door operation.

Line Following Robot – Mechatronics and Robotics Design I

What? Programmed a MiniBot to follow a black line and pick up blocks using an IR sensor. The goal was to have the MiniBot follow the line in a circle and pick up blocks three times with a minimum of 5 per time.

How? Calibrated sensor voltage readings for line detection and coded Arduino control to adjust wheel speeds to stay on the line. Used an optimized block layout to maximize pickups per run.

Results

One of two groups to pick up all 50 blocks, creating a tie for first.
